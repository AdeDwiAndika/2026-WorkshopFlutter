Repository ini adalah ruang eksperimen dan pembelajaran saya dalam mendalami pengembangan aplikasi menggunakan **Flutter**. Di sini saya mencoba berbagai konsep, widget, arsitektur, dan fitur-fitur menarik yang ditawarkan oleh Flutter.

## ✨ Fitur yang Dieksplorasi

- ✅ **Dasar Flutter & Dart** - Memahami sintaks dasar dan konsep Flutter
- ✅ **Widget Exploration** - Mencoba berbagai macam widget Flutter
- ✅ **State Management** - setState, Provider, Riverpod, Bloc
- ✅ **Navigation & Routing** - Navigasi sederhana hingga routing kompleks
- ✅ **API Integration** - Mengonsumsi REST API dengan berbagai metode
- ✅ **Local Database** - SQLite, Hive, SharedPreferences
- ✅ **Animation** - Eksplorasi animasi Flutter
- ✅ **Responsive UI** - Membuat tampilan responsif untuk berbagai ukuran layar
